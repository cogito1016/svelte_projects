## 초기 세팅

### 설치 및 실행

#### 기본실행

- npm create svelte@latest hi-svelte
- cd hi-svelte
- npm install
- npm run dev

#### 템플릿실행

###### 스벨트와 라우트가 포함된 템플릿으로 시작합니다.

- npx degit italypaleale/svelte-spa-router-template hi-svelte
- cd svelte-app

- npm run dev
